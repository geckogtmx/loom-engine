## **Purpose**

This log tracks **behavioral signals** across sessions to detect:

- tone drift
    
- reasoning shifts
    
- initiative changes
    
- role boundary issues
    
- emergent agent patterns
    

This supports:

- Drift prevention
    
- META calibration
    
- World evolution analysis
    
- Longitudinal interpretability
    

Only **four signals** + brief notes are required (KISS).

---

## **Behavior Signals (MVP)**

- **Tone Consistency:** Stable / Slight Drift / Drift
    
- **Reasoning Shape:** Structural / Narrative / Analytical / Hybrid / Fragmented
    
- **Initiative Level:** Low / Expected / High
    
- **Role Boundaries:** Clean / Minor Crossover / Drift
