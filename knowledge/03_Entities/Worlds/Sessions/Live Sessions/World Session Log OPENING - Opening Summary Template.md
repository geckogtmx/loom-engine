# 🌅 **WORLD OPENING LOG — TEMPLATE**

**Date:** `YYYY-MM-DD`  
**World:** `World Name`  
**Session ID:** `session-xxx`  
**Session Type (optional):** Exploration / Structuring / Production / Mixed  
**Operator State (optional):** If noted by Operator  
**META Status:** Ready / Adjusted / Requires Review

---

## **1. World Identity Load**

META loads:

- **World Purpose**
    
- **Tone boundaries**
    
- **Vocabulary rules**
    
- **Reasoning style**
    
- **Identity constraints** (must / must-not)
    
- **World type** (Creative, Systemic, Strategic, etc.)
    

---

## **2. Agent Team Initialization**

For each agent, META confirms:

- role
    
- tone calibration
    
- initiative level
    
- constraints
    
- readiness
    
- any pending drift flags
    

Inactive agents are parked; active roles declared.

---

## **3. Pattern Memory Snapshot**

META pulls the last session’s Pattern context:

- Last active Pattern
    
- Pending transitions
    
- Recommended starting Patterns
    
- Pattern constraints currently emphasized
    
- Cognitive mode defaults (structural, narrative, analytical)
    

---

## **4. World State Snapshot**

META surfaces:

- Last session summary
    
- Outstanding decisions
    
- Active threads (+ status)
    
- Current focus area
    
- Known bottlenecks
    
- Next-step suggestions
    
- Any World evolutions flagged (identity → drift → constraints)
    

---

## **5. Operator Preferences Loaded**

From Operator Telos:

- communication mode
    
- tone rules
    
- initiative limits
    
- challenge settings
    
- decision framing
    
- cognitive rhythm
    
- overload safeguards
    

META adjusts agent behavior accordingly.

---

## **6. Constraints & Guardrails Activated**

META applies:

- boundary constraints
    
- workflow constraints
    
- reasoning constraints
    
- cross-agent constraints
    
- behavioral guardrails
    
- context isolation safeguards
    
- no-cross-world-leakage rules
    

---

## **7. Session Initialization Summary**

A short, Operator-readable recap of:

- where we are
    
- what’s active
    
- what’s ready
    
- what’s recommended next
