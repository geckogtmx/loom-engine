
# Option Burst

> **Family:** Ideation
> **Purpose:** Rapidly generate multiple distinct options for a specific problem or decision.

## Inputs
- Problem Statement: string
- Constraints: string[] (optional)

## Steps

### 1. Diverge
**Agent:** Ideator
**Tempo:** Allegro

Generate 5-7 distinct options for the provided problem statement.
Constraints:
- No judgment or evaluation.
- Aim for variety in approach.
- Brief descriptions (1-2 sentences).

### 2. Refine
**Agent:** Critic
**Tempo:** Andante

Review the generated options.
- Merge duplicates.
- Clarify vague descriptions.
- Ensure alignment with original constraints.
Output the final list as a numbered list.
