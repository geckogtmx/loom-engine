
# Fact Check

> **Family:** Analysis
> **Purpose:** rigorously verify a set of claims against available knowledge.

## Inputs
- Claims: string
- Context: string (optional)

## Steps

### 1. Source Gathering
**Agent:** Researcher
**Tempo:** Andante

Identify key claims in the input text.
For each claim, gather potential sources or evidence relative to the World context.
List claims with their preliminary evidence status.

### 2. Verification Analysis
**Agent:** Reviewer
**Tempo:** Adagio

Analyze the evidence gathered in Step 1.
Rate each claim as Verified, False, or Unsubstantiated.
Provide a reasoning chain for each rating.
Construct a final report.
