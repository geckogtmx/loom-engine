
# Possibility Engine

> **Family:** Ideation
> **Purpose:** Generate radical new ideas by systematically removing constraints and injecting randomness.

## Inputs
- Problem Statement: string
- Current Blockers: string (optional)

## Steps

### 1. Constraint Removal (Zero Gravity)
**Agent:** Ideator
**Tempo:** Allegro

Identify the implicit and explicit constraints in the [Problem Statement].
Systematically negate them. "What if [Constraint X] did not exist?"
List the impossible scenarios that become possible.

### 2. Random Injection
**Agent:** Ideator
**Tempo:** Allegro

Inject 3 random concepts (e.g., "Mycelium", "Quantum Entanglement", "Ancient Rome").
Force-fit these concepts onto the "Zero Gravity" scenarios.
Generate hybrid ideas that merge the problem with the random stimulus.

### 3. Feasibility Mapping
**Agent:** Architect
**Tempo:** Andante

Take the wildest ideas from Step 2.
Map them back to reality. "How could we approximate this with current technology?"
Output 3 radical but potentially viable concepts.
