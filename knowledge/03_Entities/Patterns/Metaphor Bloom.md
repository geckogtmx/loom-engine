
# Metaphor Bloom

> **Family:** Ideation
> **Purpose:** Explore a complex concept by mapping it to a disparate domain to reveal hidden structure.

## Inputs
- Concept: string
- Target Domain: string (optional)

## Steps

### 1. Seed
**Agent:** Ideator
**Tempo:** Allegro

Identify the core characteristics and dynamics of the input [Concept].
If a [Target Domain] is provided, analyze its structural elements; otherwise, propose 3 distinct target domains (e.g., Biology, Architecture, Occultism).

### 2. Cultivate
**Agent:** Ideator
**Tempo:** Andante

Map the elements of the [Concept] onto the [Target Domain].
Push the metaphor until it breaks. Where does the mapping fail? These failures are insights.
Describe the system dynamics in the language of the target domain.

### 3. Harvest
**Agent:** Illustrator
**Tempo:** Adagio

translate the metaphorical insights back into the source domain.
What new relationships or functions were revealed?
Output a list of novel perspectives derived from the metaphor.
