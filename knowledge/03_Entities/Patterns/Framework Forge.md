
# Framework Forge

> **Family:** Structuring
> **Purpose:** Systematize unstructured information into a coherent mental model or framework.

## Inputs
- Topic: string
- Raw Material: string (optional)

## Steps

### 1. Deconstruct
**Agent:** Analyst
**Tempo:** Allegro

Break down the [Topic] and [Raw Material] into atomic concepts.
Discard noise and redundancy.
Cluster related atoms into "Heaps".

### 2. Model Selection
**Agent:** Architect
**Tempo:** Andante

Review the Heaps from Step 1.
Propose 3 standard frameworks (e.g., 2x2 Matrix, Hierarchy, Flywheel, Funnel) that might fit the data.
Select the best fit, or invent a custom hybrid structure.

### 3. Synthesize
**Agent:** Architect
**Tempo:** Adagio

Populate the selected framework with the clustered concepts.
Name the quadrants, stages, or layers.
Define the relationships (arrows/flows) between components.
Output the definition of the new Framework.
